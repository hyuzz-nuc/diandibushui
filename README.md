# 点滴补水 (DropHydrate) 💧

> 每一滴水，都是对身体的宠爱。

---

## 🌟 这是个啥？

**点滴补水**是一款微信小程序，专门治"忙起来就忘记喝水"的毛病。

你是不是也这样：
- 对着电脑一坐就是半天，口渴了才想起来喝水
- 明明知道每天要喝 8 杯水，但总是记不住
- 想养成喝水习惯，但坚持不了几天

这个小工具就是来帮你的：**定时提醒你喝水，记录你喝了多少，还能拉上好友互相监督**。不是那种冷冰冰的工具，而是像个贴心的小伙伴，陪你一起养成健康习惯。

---

## ✨ 能幹啥？

### 🥤 喝水记录
- 首页超大进度条，一眼看到今天喝了多少
- 快捷按钮：+100ml、+200ml、+300ml，点一下就行
- 也能自定义输入任意毫升数

### ⏰ 智能提醒
- 每天定时推送，到点提醒你喝水
- 可以自定义提醒时间和频率
- 早上 9 点到晚上 9 点，覆盖你的清醒时间

### 📊 数据统计
- 周视图、月视图，看看自己的喝水习惯
- 打卡日历，连续达标有高亮标记
- 连续打卡天数、达标率，一目了然

### 👯 好友监督
- 绑定好友，互相查看喝水进度
- 可以给朋友发提醒："快去喝水！"
- 一个人坚持不了？那就两个人一起

### 🏆 称号系统
- 连续打卡 3 天 → **饮水萌新**
- 连续打卡 7 天 → **饮水达人**
- 连续打卡 15 天 → **饮水王者**
- 连续打卡 30 天 → **饮水大神**

### 💡 健康文章推荐（开发中）
- 针对口腔溃疡、上火、熬夜等常见问题
- 推荐对应的饮水方案和养生文章
- 从文章页直接调整今日喝水目标

### 🤝 智能搭子（开发中）
- 根据喝水习惯匹配相似的"喝水搭子"
- 一起打卡，互相监督
- 排行榜看看谁最坚持

---

## 🛠️ 技术栈

- **前端**：微信原生小程序（WXML/WXSS/JS）
- **UI 组件**：Vant Weapp
- **图表**：wx-charts
- **后端**：微信云开发（Serverless，不用自己搭服务器）
- **AI 健康顾问**：智谱 AI GLM-4-Flash（养生问题随时问）

## 🚀 怎么用？

### 开发者
```bash
# 1. 克隆项目
git clone git@github.com:hyuzz-nuc/diandibushui.git
cd diandibushui

# 2. 安装依赖
npm install

# 3. 用微信开发者工具打开项目目录

# 4. 在微信开发者工具中：
#    - 项目配置 → 勾选"使用云开发"
#    - 上传云函数（右键 cloudfunctions 目录）
#    - 编译预览
```

### 普通用户
1. 微信搜索"点滴补水"小程序（上线后）
2. 首次使用授权登录
3. 设置你的每日喝水目标（默认 2000ml）
4. 开启提醒，然后等着被"催喝水"就行
