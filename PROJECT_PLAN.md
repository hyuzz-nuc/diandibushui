## 1. 产品概述 (Product Overview)

**产品名称**: 点滴补水 (DropHydrate)  
**Slogan**: 每一滴水，都是对身体的宠爱。  
**定位**: 一款轻量级、高颜值的饮水提醒工具。不仅帮助用户养成健康的饮水习惯，更通过“好友监督”和“称号激励”机制，让喝水变得有趣且不孤单。  
**核心价值**: 
- **工具价值**: 极致简单的记录体验，精准的定时提醒。
- **情绪价值**: 萌趣的称号体系，好友间的温暖互动（或“严厉”监督）。
- **知识价值**: 智能推荐健康文章，解决日常小症状（如口腔溃疡、皮肤干燥）。

## 2. 功能清单与优先级 (Feature List & Priority)

我们将开发过程分为三个阶段，优先保证核心闭环。

### P0：核心生存期 (Core MVP) - *已完成*
*目标：用户能用，能记，能看。*

| 模块 | 功能点 | 详细说明 |
| :--- | :--- | :--- |
| **初始化** | 个性化配置 | 引导设置每日目标(默认2000ml)、提醒时段(9:00-21:00)、提醒间隔(1h)。授权订阅消息。 |
| **喝水记录** | 快捷记录 | 首页提供 200/300/500ml 快捷气泡按钮。 |
| | 自定义记录 | 支持输入任意数值（ml）。 |
| | 进度反馈 | 首页大圆环/水波纹进度条实时更新，展示今日已喝水量/目标量。 |
| **定时提醒** | 基础提醒 | 每日默认9:00和14:00推送（基于订阅消息）。支持自定义时间。 |
| **数据统计** | 趋势图表 | 简单的周/月柱状图，展示每日饮水量。 |
| **用户系统** | 一键登录 | 微信授权快捷登录，获取OpenID。 |

### P1：社交与激励期 (Social & Gamification) - *已完成*
*目标：用户爱用，有粘性。*

| 模块 | 功能点 | 详细说明 |
| :--- | :--- | :--- |
| **好友监督** | 邀请绑定 | 生成专属邀请卡/链接，好友点击确认后建立双向绑定关系。 |
| | 监督看板 | 查看好友今日进度。 |
| | 互动消息 | 发送“快去喝水”、“太棒了”等模板消息（小程序内通知/震动）。 |
| **打卡体系** | 每日打卡 | 达标即自动打卡，中断则重置连续天数。 |
| | 称号系统 | 连续3天(萌新) -> 7天(达人) -> 15天(王者) -> 30天(大神)。 |
| | 个人主页 | 展示当前称号、连续打卡天数。 |
| **提醒升级** | 喝水搭子提醒 | 每天2次机会，强制提醒好友（需间隔2小时），支持自定义文案。 |

### P2：体验优化与裂变期 (Polishing & Growth) - *已完成*
*目标：极致体验，用户传播。*

| 模块 | 功能点 | 详细说明 |
| :--- | :--- | :--- |
| **数据增强** | 导出分享 | 统计图表支持生成长图保存/分享。 |
| **激励彩蛋** | 成就弹窗 | 获得新称号/连续打卡7天时，弹出精美全屏贺卡。 |
| **辅助功能** | 智能推荐 | 根据体重/天气推荐饮水目标。 |
| | 深色模式 | 完美适配iOS/Android深色模式。 |

### P3：智能推荐与社区期 (Smart Recommendation & Community) - *开发中*
*目标：知识赋能，社交增强。*

| 模块 | 功能点 | 详细说明 |
| :--- | :--- | :--- |
| **发现 (Discovery)** | 智能文章推荐 | 针对日常症状（如口腔溃疡、便秘、皮肤干）推荐饮水/饮食调理文章。 |
| | 症状自查 | 简单的标签选择（如“上火”、“熬夜”），推荐今日饮水方案。 |
| **社交 (Social)** | 智能搭子推荐 | 根据饮水习惯/打卡频率，推荐相似的用户作为“喝水搭子”（需双方同意）。 |
| | 互动排行榜 | 好友间周/月打卡排行榜。 |

## 3. UI/UX 设计规范 (Design Specification)

**设计理念**: 像水一样清透、流动、无负担。

### 3.1 配色方案 (Color Palette)
*   **主色 (Primary)**: `#00B0FF` (清透蓝) - 用于主按钮、进度条高亮、关键数据。
*   **辅助色 (Secondary)**: `#4FC3F7` (浅天蓝) - 用于次级按钮、背景渐变。
*   **背景色 (Background)**: `#F0F8FF` (爱丽丝蓝) / `#FFFFFF` - 保持页面干净。
*   **深色模式背景**: `#121212` (深灰) / `#1f1f1f` (卡片灰)。
*   **警告色 (Warning)**: `#FF9800` (橙色) - 用于未达标提示。
*   **文本色**: 
    *   标题 `#333333` (深色: `#e0e0e0`)
    *   正文 `#666666` (深色: `#a0a0a0`)
    *   辅助 `#999999` (深色: `#666666`)

### 3.2 核心页面布局 (Wireframes) - *V1.1 更新*

**TabBar 结构调整**:
1.  **发现 (Discovery)**: 左一，新增。智能文章推荐入口。
2.  **社交 (Social)**: 左二，新增。好友列表、搭子推荐、排行榜。
3.  **首页 (Home)**: 中间（核心）。喝水记录大圆环。
4.  **统计 (Stats)**: 右二。日历与趋势图。
5.  **我的 (Profile)**: 右一。个人信息与设置。

#### A. 首页 (Home) - *不变*
*   **顶部**: 简单的问候语（"早上好，记得喝水哦"）+ 天气图标（可选）。
*   **中部 (核心)**: 
    *   巨大的动态水球/圆环，水位随进度升高。
    *   中央显示：`已喝 1200ml` / `目标 2000ml`。
*   **下部**: 
    *   一排快捷按钮：`[+100ml]` `[+200ml]` `[+300ml]` `[自定义]`。
    *   记录成功时，水球有波动动画 + "咕噜"音效。

#### B. 发现 (Discovery) - *New*
*   **顶部**: “症状自查”标签云（上火、感冒、熬夜...）。
*   **列表**: 瀑布流展示健康文章卡片（标题 + 摘要 + 封面图）。
*   **详情**: 点击进入文章详情页，底部有“打卡今日推荐水量”快捷按钮。

#### C. 社交 (Social) - *New*
*   **我的好友**: 现有好友列表（头像、进度、提醒按钮）。
*   **推荐搭子**: “发现新朋友”卡片，展示活跃度相似的陌生人（昵称脱敏），点击申请添加。
*   **排行榜**: 本周打卡天数排行。

#### D. 统计 (Stats) - *已优化*
*   **日历视图**: 月份切换，打卡成功的日期高亮。
*   **趋势图**: 近7天饮水柱状图（带日期）。
*   **核心数据**: 连续打卡天数、本月达标率、日均饮水量。

#### E. 个人主页 (Profile) - *已优化*
*   **头部**: 卡片式设计，展示头像、昵称、**当前称号**（可点击生成海报）。
*   **功能列表**: 邀请好友、提醒设置、关于我们。

## 4. 技术架构与数据库设计 (Technical Architecture)

**技术栈**: 
*   前端: 微信原生 WXML/WXSS/JS (Canvas 2D)
*   UI库: Vant Weapp (按钮、弹窗、表单)
*   图表: wx-charts (轻量级)
*   后端: 微信云开发 (Cloud Functions + Cloud Database)

### 4.1 数据库集合 (Database Collections)

1.  **users (用户表)**
    ```json
    {
      "_id": "openid_xxx",
      "nickname": "小明",
      "avatar_url": "...",
      "daily_goal": 2000,
      "current_title": "饮水萌新",
      "consecutive_days": 5,
      "total_days": 12,
      "last_checkin_date": "2026-01-22",
      "tags": ["熬夜", "运动"], // 用户标签，用于推荐
      "settings": { ... }
    }
    ```

2.  **records (饮水记录表)** - *不变*

3.  **checkins (打卡流水表)** - *新增，防重*
    ```json
    {
      "_id": "openid_2026-01-22", // 唯一主键
      "_openid": "user_openid",
      "date_str": "2026-01-22",
      "createTime": ...
    }
    ```

4.  **articles (文章表)** - *New*
    ```json
    {
      "_id": "article_id",
      "title": "口腔溃疡怎么办？多喝水能缓解吗？",
      "summary": "...",
      "cover_img": "...",
      "content": "...", // 富文本或Markdown
      "tags": ["口腔溃疡", "上火"],
      "recommend_amount": 2500 // 推荐饮水量
    }
    ```

5.  **relations (好友关系表)** - *不变*
